Introduction

This tool is developed for the lecturers where they can upload their own document in the repository. This system is build using Django and MongoDB as database. This system also used NLTK and BeautifulSoup 4. NLTK library one of the leading natural language procesing toolkit develpoed using python language. It is used for tokenizing of words and sentences. While BeautifulSoup is use to extract website content.

How to use the system

User needs to have an account. After loging in user can see options for creating the class and uploading document. By entering necessary information user can create a class. System will provide the user with a unique ID which is necessary because the sudent will need to enter that ID for submission. Clicking on the ID will show the view of submitted document. If user wish to delete any document or class or document from class, it can be done easily as it is all time visible in action column
